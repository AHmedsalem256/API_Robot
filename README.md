API Testing with Robot Framework

This is a simple project demonstrating API testing using Robot Framework with the RequestsLibrary. The project covers testing of three different APIs to perform basic operations such as GET, POST, and DELETE requests.

Features

Automated API testing using Robot Framework

Utilizes RequestsLibrary for HTTP requests

Example test cases for multiple API endpoints

Easy to extend for more APIs and test scenarios

Ideal for beginners learning API automation

Getting Started

Install Python (3.11+ recommended)

Install Robot Framework:

pip install robotframework

Install RequestsLibrary:

pip install robotframework-requests

Run the test cases:

robot tests/

Test Cases Included

Add Book API – POST request example

Get Book API – GET request example

Delete Book API – DELETE request example
